\\Formula-:
$$ J.C =  \frac {n(A \cap B)} {n(A \cup B)}  $$
Value of Jaccard Coffecient varies from 0 to 1. The value 0 means no similarity and value 1 means completely similar. To find the distance we used the formula $1 - J.C$
